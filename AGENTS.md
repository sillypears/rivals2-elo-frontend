- **Components**: PascalCase naming, .jsx extension
- **Styling**: Tailwind CSS with custom design tokens
- **UI Library**: Radix UI components via shadcn/ui
- **Error Handling**: Throw errors in API functions, handle in components
- **Formatting**: ESLint with recommended rules + React hooks plugin
- **Naming**: camelCase for files/utilities, PascalCase for components</content>
